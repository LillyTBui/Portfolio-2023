# Portfolio

## Description

Personal portfolio website made in 2022.

## Built With

- HTML
- CSS/SASS
- JavaScript
- React
- React Bootstrap

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:LillyTBui/portfolio2023.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
